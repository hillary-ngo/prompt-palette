# The Prompt Palette 🎨

A curated library of visual design styles with copy-ready prompts for generating and styling slide decks.

**Curated by Hillary Uyen Le Ngo**

---

## What is this?

The Prompt Palette is a reference tool for educators, designers, and presenters who want to create visually consistent, well-designed slide decks using AI tools (such as Google Gemini, ChatGPT, or Claude).

Each style card includes:
- A visual color palette preview
- Mood and aesthetic description
- Recommended font pairings
- A copy-ready prompt you can paste directly into any AI tool

## How to use

1. Open the library: [hillary-ngo.github.io/prompt-palette](https://hillary-ngo.github.io/prompt-palette)
2. Browse or search for a style that fits your presentation
3. Click a card to expand it and reveal the full prompt
4. Click **Copy prompt** and paste it into your AI tool of choice
5. Describe your content — the AI will apply the style to your slides

## Want to add a style?

Open `index.html`, find the `styles` array near the bottom of the file, and add a new entry following the same format as the existing ones. Commit the change and the live site will update within a minute or two.
